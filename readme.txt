This a simple to use WordPress plugin that let you to set another RSS Feed Url to be displayed for one or a list of many IP's.

== Description ==

The main reason i've made this plugin is to block those who want to copy the contents of WordPress's default RSS Feed.

== Installation ==

1. Upload `block-rss-reading.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's it!
